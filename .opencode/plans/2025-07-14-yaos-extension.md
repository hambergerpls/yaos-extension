# YAOS Extension Plugin — Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build an Obsidian plugin (`yaos-extension`) that extends YAOS to show collaborator names on remote cursors and display a "who's online" presence indicator — without forking YAOS.

**Architecture:** The plugin accesses YAOS internals at runtime via `app.plugins.getPlugin("yaos")` to reach the Yjs `Awareness` instance. It subscribes to awareness `change` events to track peer joins/leaves/updates. Two features: (1) CSS override to unhide `.cm-ySelectionInfo` name labels on cursors, and (2) a status bar item + hover dropdown showing connected collaborators. Graceful fallbacks if YAOS is not installed or not yet initialized.

**Tech Stack:** Obsidian API, TypeScript, esbuild, Yjs Awareness (accessed at runtime through YAOS, not bundled)

```ts
import { App } from "obsidian";

/** Minimal type for the Awareness object from y-protocols. */
export interface AwarenessLike {
  clientID: number;
  states: Map<number, Record<string, unknown>>;
  getStates(): Map<number, Record<string, unknown>>;
  getLocalState(): Record<string, unknown> | null;
  on(event: "change" | "update", handler: (data: AwarenessChangeData, origin: unknown) => void): void;
  off(event: "change" | "update", handler: (data: AwarenessChangeData, origin: unknown) => void): void;
}

export interface AwarenessChangeData {
  added: number[];
  updated: number[];
  removed: number[];
}

/** Shape of a user field in awareness state. */
export interface AwarenessUser {
  name: string;
  color: string;
  colorLight: string;
}

/** Represents a remote collaborator detected via Awareness. */
export interface RemotePeer {
  clientId: number;
  name: string;
  color: string;
  colorLight: string;
  hasCursor: boolean;
}

/**
 * Attempts to obtain the YAOS plugin instance.
 * Returns null if YAOS is not installed or not enabled.
 */
function getYaosPlugin(app: App): unknown | null {
  try {
    return (app as any).plugins?.getPlugin("yaos") ?? null; // eslint-disable-line @typescript-eslint/no-explicit-any
  } catch {
    return null;
  }
}

/**
 * Attempts to extract the Awareness instance from YAOS internals.
 * Returns null if the access path is broken (YAOS version change, etc.).
 *
 * Access path: VaultCrdtSyncPlugin → vaultSync → provider → awareness
 */
export function getAwareness(app: App): AwarenessLike | null {
  const plugin = getYaosPlugin(app);
  if (!plugin) return null;

  const vaultSync = (plugin as any)?.vaultSync; // eslint-disable-line @typescript-eslint/no-explicit-any
  if (!vaultSync) return null;

  const provider = vaultSync.provider;
  if (!provider) return null;

  const awareness = provider.awareness;
  if (!awareness || typeof awareness.getStates !== "function") return null;

  return awareness as AwarenessLike;
}

/** Checks whether YAOS is installed and enabled. */
export function isYaosAvailable(app: App): boolean {
  return getYaosPlugin(app) !== null;
}

/** Checks whether YAOS's sync connection appears active. */
export function isYaosConnected(app: App): boolean {
  const plugin = getYaosPlugin(app);
  if (!plugin) return false;
  const vaultSync = (plugin as any)?.vaultSync; // eslint-disable-line @typescript-eslint/no-explicit-any
  if (!vaultSync) return false;
  const provider = vaultSync.provider;
  if (!provider) return false;
  return !!provider.wsconnected;
}

/** Returns the list of remote peers from Awareness, excluding the local client. */
export function getRemotePeers(awareness: AwarenessLike): RemotePeer[] {
  const localClientId = awareness.clientID;
  const peers: RemotePeer[] = [];

  for (const [clientId, state] of awareness.getStates()) {
    if (clientId === localClientId) continue;
    const user = state.user as AwarenessUser | undefined;
    const cursor = state.cursor as { anchor: unknown; head: unknown } | null | undefined;

    peers.push({
      clientId,
      name: user?.name || "Anonymous",
      color: user?.color || "#30bced",
      colorLight: user?.colorLight || "#30bced33",
      hasCursor: cursor != null && cursor.anchor != null && cursor.head != null,
    });
  }

  return peers;
}

/** Returns the local device name from YAOS settings, or a fallback. */
export function getLocalDeviceName(app: App): string {
  const plugin = getYaosPlugin(app);
  if (!plugin) return "Unknown";
  const settings = (plugin as any)?.settings; // eslint-disable-line @typescript-eslint/no-explicit-any
  return settings?.deviceName || "Unknown";
}
```

import { App } from "obsidian";
import {
  getAwareness,
  isYaosAvailable,
  getRemotePeers,
  type AwarenessLike,
  type AwarenessChangeData,
  type RemotePeer,
} from "./yaosApi";

export type PresenceChangeCallback = (
  peers: RemotePeer[],
  awareness: AwarenessLike
) => void;

/**
 * Tracks remote collaborators by subscribing to YAOS's Awareness instance.
 * Handles the async wait for YAOS to initialize and provides a clean
 * subscribe/unsubscribe API.
 */
export class PresenceTracker {
  private app: App;
  private awareness: AwarenessLike | null = null;
  private handler: ((data: AwarenessChangeData, origin: unknown) => void) | null = null;
  private callback: PresenceChangeCallback | null = null;
  private pollingInterval: number | null = null;
  private initialized = false;

  constructor(app: App) {
    this.app = app;
  }

  /**
   * Start tracking. Will poll for YAOS availability until found,
   * then subscribe to Awareness change events.
   */
  start(onChange: PresenceChangeCallback): void {
    this.callback = onChange;
    this.tryConnect();
  }

  /** Stop tracking. Cleans up all listeners and intervals. */
  stop(): void {
    this.detachFromAwareness();
    if (this.pollingInterval !== null) {
      window.clearInterval(this.pollingInterval);
      this.pollingInterval = null;
    }
    this.callback = null;
    this.initialized = false;
  }

  /**
   * Attempt to connect to YAOS Awareness.
   * If YAOS isn't ready yet, starts a polling interval.
   */
  private tryConnect(): void {
    if (!isYaosAvailable(this.app)) {
      // YAOS not installed or not enabled. Poll until it appears.
      this.pollingInterval = window.setInterval(() => {
        if (isYaosAvailable(this.app)) {
          if (this.pollingInterval !== null) {
            window.clearInterval(this.pollingInterval);
            this.pollingInterval = null;
          }
          this.tryConnect();
        }
      }, 2000);
      return;
    }

    const awareness = getAwareness(this.app);
    if (!awareness) {
      // YAOS is installed but sync isn't initialized yet (user hasn't configured host/token).
      // Poll until it becomes available.
      this.pollingInterval = window.setInterval(() => {
        const aw = getAwareness(this.app);
        if (aw) {
          if (this.pollingInterval !== null) {
            window.clearInterval(this.pollingInterval);
            this.pollingInterval = null;
          }
          this.attachToAwareness(aw);
        }
      }, 3000);
      return;
    }

    this.attachToAwareness(awareness);
  }

  private attachToAwareness(awareness: AwarenessLike): void {
    this.awareness = awareness;
    this.initialized = true;

    this.handler = (_data: AwarenessChangeData, _origin: unknown) => {
      this.notifyCallback();
    };

    awareness.on("change", this.handler);

    // Fire initial state
    this.notifyCallback();
  }

  private detachFromAwareness(): void {
    if (this.handler && this.awareness) {
      try {
        this.awareness.off("change", this.handler);
      } catch {
        // Awareness may have been destroyed
      }
    }
    this.handler = null;
  }

  private notifyCallback(): void {
    if (this.callback && this.awareness) {
      const peers = getRemotePeers(this.awareness);
      this.callback(peers, this.awareness);
    }
  }

  /** Whether the tracker has successfully connected to YAOS. */
  get isReady(): boolean {
    return this.initialized;
  }

  /** Current awareness instance if connected. */
  get currentAwareness(): AwarenessLike | null {
    return this.awareness;
  }
}

import { type RemotePeer } from "./yaosApi";
import { type YaosExtensionSettings } from "./settings";

/**
 * Manages the status bar item showing connected peers and sync status.
 */
export class PresenceStatusBar {
  private statusBarEl: HTMLElement;
  private settings: YaosExtensionSettings;
  private tooltipEl: HTMLElement | null = null;

  constructor(statusBarEl: HTMLElement, settings: YaosExtensionSettings) {
    this.statusBarEl = statusBarEl;
    this.settings = settings;
    this.statusBarEl.addClass("yaos-extension-statusbar");
  }

  /**
   * Update the status bar with current peer info.
   */
  update(peers: RemotePeer[], isConnected: boolean, _localDeviceName: string): void {
    this.statusBarEl.empty();

    if (!isConnected && peers.length === 0) {
      this.renderDisconnected();
      return;
    }

    // Connection dot
    const dot = this.statusBarEl.createSpan({ cls: "yaos-extension-dot connected" });
    dot.setAttribute("aria-label", isConnected ? "Sync connected" : "Sync disconnected");

    if (peers.length === 0 && isConnected) {
      this.statusBarEl.createSpan({ text: "1 device online" });
      return;
    }

    // Peer count text
    if (peers.length === 1) {
      this.statusBarEl.createSpan({ text: `You + ${peers[0].name}` });
    } else {
      this.statusBarEl.createSpan({ text: `You + ${peers.length} collaborators` });
    }

    // Peer dots
    if (this.settings.showPeerDotsInStatusBar) {
      const peersContainer = this.statusBarEl.createSpan({ cls: "yaos-extension-peers" });
      for (const peer of peers) {
        const peerDot = peersContainer.createSpan({ cls: "yaos-extension-peer-dot" });
        peerDot.style.backgroundColor = peer.color;
        peerDot.setAttribute("aria-label", peer.name);
        peerDot.addEventListener("mouseenter", (evt) => {
          this.showTooltip(evt, peer.name, peer.color, peer.hasCursor);
        });
        peerDot.addEventListener("mouseleave", () => {
          this.hideTooltip();
        });
      }
    }
  }

  private renderDisconnected(): void {
    const dot = this.statusBarEl.createSpan({ cls: "yaos-extension-dot disconnected" });
    dot.setAttribute("aria-label", "Sync disconnected");
    this.statusBarEl.createSpan({ text: "Not synced" });
  }

  private showTooltip(
    evt: MouseEvent,
    name: string,
    color: string,
    hasCursor: boolean
  ): void {
    this.hideTooltip();
    this.tooltipEl = document.body.createDiv({ cls: "yaos-extension-tooltip" });
    const dot = this.tooltipEl.createSpan({ cls: "yaos-extension-peer-dot" });
    dot.style.backgroundColor = color;
    this.tooltipEl.createSpan({ text: ` ${name}` });
    if (hasCursor) {
      this.tooltipEl.createSpan({ text: " ✎", attr: { style: "opacity:0.6" } });
    }
    // Position near the mouse
    this.tooltipEl.style.left = `${evt.clientX + 10}px`;
    this.tooltipEl.style.top = `${evt.clientY - 30}px`;
  }

  private hideTooltip(): void {
    if (this.tooltipEl) {
      this.tooltipEl.remove();
      this.tooltipEl = null;
    }
  }

  destroy(): void {
    this.hideTooltip();
    this.statusBarEl.empty();
  }
}

import { Notice, Plugin, PluginSettingTab, App, Setting } from "obsidian";
import {
  DEFAULT_SETTINGS,
  type YaosExtensionSettings,
} from "./settings";
import { PresenceTracker } from "./presenceTracker";
import { PresenceStatusBar } from "./statusBar";
import { isYaosAvailable, isYaosConnected, getLocalDeviceName, getRemotePeers, type RemotePeer } from "./yaosApi";

export default class YaosExtensionPlugin extends Plugin {
  settings: YaosExtensionSettings = DEFAULT_SETTINGS;
  tracker: PresenceTracker | null = null;
  statusBar: PresenceStatusBar | null = null;
  statusBarEl: HTMLElement | null = null;

  async onload() {
    await this.loadSettings();

    // Toggle cursor names CSS class based on setting
    this.applyCursorNames();

    // Status bar item
    this.statusBarEl = this.addStatusBarItem();
    this.statusBar = new PresenceStatusBar(this.statusBarEl, this.settings);

    if (!this.settings.showStatusBar) {
      this.statusBarEl.style.display = "none";
    }

    // Check if YAOS is available
    if (!isYaosAvailable(this.app)) {
      this.statusBar.update([], false, "");
      new Notice("YAOS Extension: YAOS plugin not found. Please install and enable YAOS first.");
    } else {
      // Start presence tracking
      this.tracker = new PresenceTracker(this.app);
      this.tracker.start((peers, _awareness) => {
        this.onPresenceChange(peers);
      });
    }

    // Settings tab
    this.addSettingTab(new YaosExtensionSettingTab(this.app, this));

    // Re-check YAOS availability periodically (in case it's loaded after us)
    this.registerInterval(
      window.setInterval(() => {
        if (!isYaosAvailable(this.app) && this.tracker) {
          this.tracker.stop();
          this.tracker = null;
          this.statusBar?.update([], false, "");
        } else if (isYaosAvailable(this.app) && !this.tracker) {
          this.tracker = new PresenceTracker(this.app);
          this.tracker.start((peers, _awareness) => {
            this.onPresenceChange(peers);
          });
        }
      }, 5000)
    );
  }

  onunload() {
    this.tracker?.stop();
    this.statusBar?.destroy();
    document.body.classList.remove("yaos-extension-names");
    document.querySelectorAll(".yaos-extension-tooltip").forEach((el) => el.remove());
  }

  async loadSettings() {
    this.settings = Object.assign({}, DEFAULT_SETTINGS, await this.loadData());
  }

  async saveSettings() {
    await this.saveData(this.settings);
  }

  applyCursorNames() {
    if (this.settings.showCursorNames) {
      document.body.classList.add("yaos-extension-names");
    } else {
      document.body.classList.remove("yaos-extension-names");
    }
  }

  private onPresenceChange(peers: RemotePeer[]) {
    const isConnected = isYaosConnected(this.app);
    const localName = getLocalDeviceName(this.app);
    this.statusBar?.update(peers, isConnected, localName);
  }
}

## Key Design Decisions

1. **CSS class toggle** for cursor names rather than DOM manipulation. YAOS adds `.vault-crdt-show-cursors` to `<body>` when cursors are enabled. Our plugin adds `.yaos-extension-names`. The combined selector `.vault-crdt-show-cursors.yaos-extension-names .cm-ySelectionInfo` overrides YAOS's `display: none !important`.

2. **Name labels on hover only** — following y-codemirror.next's default convention. This avoids the "looks like inserted text" problem that caused YAOS to hide names in the first place.

3. **Polling + event hybrid** — PresenceTracker polls for YAOS availability (since YAOS may load after us), then subscribes to Awareness `change` events once connected. This provides responsive updates without continuous polling.

4. **Graceful degradation** — If YAOS is not installed, the plugin shows a notice and a disconnected indicator. If YAOS updates break the access path, the plugin degrades to showing "Not synced" rather than crashing.

5. **No Yjs dependency bundled** — The plugin never imports Yjs directly. It accesses the Awareness instance through YAOS at runtime, avoiding version conflicts.
